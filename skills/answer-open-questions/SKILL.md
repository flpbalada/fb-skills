# Answer Open Questions

- Answer all questions in one response.
- Give one to three distinct answers per question.
- Recommend exactly one answer with a short reason.
- State only assumptions that could change the recommendation.

## Output

```md
### [Question]

- **[Answer] (Recommended):** [Short reason]
- **[Answer]**
- **[Answer]**
```
